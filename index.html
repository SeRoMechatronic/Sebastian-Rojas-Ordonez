Plug-and-Play LLM Knowledge Extraction for Robot Navigation
Published
Machine Learning and Knowledge Extraction (MDPI MAKE) · Feb 2026

Fine-tuning-free edge framework leveraging Large Language Models deployed on resource-constrained embedded hardware (NVIDIA Jetson) to enhance autonomous robot navigation via zero-shot prompt engineering. Demonstrates real-time LLM inference at the edge without cloud dependency, enabling knowledge-augmented robotic systems in unstructured environments.

Edge AI LLMs Robot Navigation NVIDIA Jetson Zero-Shot
Google Scholar
Natural-Language Relay Control for a SISO Thermal Plant
Published
Applied Sciences (MDPI) · Dec 2025

Proof-of-concept architecture coupling a Large Language Model to an industrial relay controller for a SISO thermal plant. Validated against a conventional hysteresis controller, the approach demonstrates LLM-based relay control as a viable paradigm in cyber-physical systems, with natural-language setpoint specification and explainable control decisions.
